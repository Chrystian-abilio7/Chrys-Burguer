## ✨ About the Project

Chrys Burguer is an interactive menu where users can explore products, apply discounts, filter vegan options, and calculate the total price.

This project was built to practice **real JavaScript concepts applied to a visual interface**.

---

## 🚀 Features

* 🍔 Show and hide menu items
* 💸 Apply 10% discount using **map()**
* 🧮 Calculate total price with **reduce()**
* 🌱 Filter vegan items using **filter()**
* 🎨 Smooth UI with hover effects and transitions

---

## 🧠 Concepts Applied

* DOM manipulation
* Array methods (`map`, `filter`, `reduce`, `forEach`)
* Event handling
* Dynamic UI updates
